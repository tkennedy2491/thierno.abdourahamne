# **App Name**: DevSphere Portfolio

## Core Features:

- Dynamic Hero Section: Engaging hero section with animated text (typing effect) for name and title, including a prominent call-to-action button.
- About Me Section: A dedicated section to introduce the portfolio owner, their professional background, and personal interests.
- Skills and Technologies Display: Visually appealing presentation of programming languages, frameworks, and tools used by the portfolio owner.
- Project Showcase Gallery: Interactive gallery displaying key projects with images, brief descriptions, and links to live demos or repositories.
- Contact Form: A functional contact form enabling visitors to send direct messages or inquiries to the portfolio owner.
- Smooth Section Navigation: Intuitive navigation bar with smooth scrolling to different sections of the single-page application.
- AI-Powered Inquiry Assistant: A tool that provides visitors with AI-generated suggestions to start their contact messages, personalizing based on the portfolio content.

## Style Guidelines:

- Color Anchor: 'Modern digital identity' with a 'night mode' aesthetic to evoke creativity and focus. Dark color scheme. Primary color: A vibrant, deep violet (`#5C2EE6`) for interactive elements and highlights, suggesting innovation and energy. Background color: A very dark, almost black hue with a subtle hint of purple (`#1C1A1F`), providing a clean, minimalist canvas for content. Accent color: A cool blue-purple (`#6C74EB`), analogous to the primary, used for secondary highlights and creating visual contrast without clashing.
- Headline and body font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics suitable for a technical portfolio.
- Utilize clean, vector-based SVG icons for social media links, skill representations, and action buttons, maintaining a sleek and professional appearance consistent with modern digital design.
- A responsive single-page application layout, designed for optimal viewing across various devices. The content is organized into distinct, scrollable sections (Hero, About, Skills, Projects, Contact) for easy navigation and a cohesive user experience.
- Subtle, non-intrusive animations, including smooth scrolling transitions between sections, tasteful fade-in effects for content on scroll, and engaging hover effects for interactive components like project cards and buttons to enhance user engagement.
